# Jiggy Capital Main Website

This is the main public-facing website for jiggycapital.com.

## Files

- `index.html` - Main website HTML
- `script.js` - Website JavaScript logic
- `styles.css` - Website styling
- `utils.js` - Utility functions
- `favicon.svg` - Site favicon
- `golf-photo.jpg` - Profile photo

## Features

- Portfolio overview
- Performance metrics
- Allocation charts
- Company news
- Upcoming events

## Note

The advanced financial analysis dashboard is now available at [app.jiggycapital.com](https://app.jiggycapital.com) (Next.js application in the `/app` folder).
