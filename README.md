# Jiggy Capital

Portfolio and financial analysis platform.

## Structure

- `/website` - Main public website (jiggycapital.com)
  - Portfolio overview, performance metrics, allocation charts
  - Built with vanilla HTML/CSS/JavaScript
  
- `/app` - Financial Analysis Dashboard (app.jiggycapital.com)
  - Advanced data table and charting
  - Built with Next.js 16, TypeScript, TanStack Table, Recharts
  - Fully customizable columns from Google Sheets

## Deployment

- Main website: Deployed via GitHub Pages or similar
- Analysis app: Deployed on Vercel at app.jiggycapital.com

## Data Sources

All data is fetched from Google Sheets:
- Positions Detailed
- Watchlist Detailed
- Performance metrics
- Company logos
